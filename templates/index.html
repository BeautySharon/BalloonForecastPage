🎈 Balloon Trajectory Prediction Project

Machine Learning-Based Balloon Trajectory Prediction & Airspace Safety System

This project builds a comprehensive real-time balloon trajectory prediction and airspace safety monitoring system using live data from Windborne Systems and the OpenSky Network.

  • 📊 Data Processing: Retrieves 24-hour balloon position data via APIs, interpolates missing timestamps, and applies the Haversine formula for speed calculation.
  • 🤖 Machine Learning: Utilizes a Random Forest Regressor to predict future balloon speed & direction, incorporating wind speed analysis.
  • ✈️ Collision Detection: Implements a KD-tree algorithm to identify potential airspace hazards between balloons and aircraft.
  • 🌍 Interactive Visualization: Features a real-time map displaying historical trajectories, predicted paths, and potential collision zones.
  • 🔎 Applications: Meteorology, unmanned balloon navigation, and airspace safety management.
