Live Balloon Tracking & Airspace Safety System

This project builds a comprehensive real-time balloon tracking, trajectory prediction, and airspace safety monitoring system by integrating live data from Windborne Systems, OpenSky Network, and OpenWeather API. The system dynamically updates every hour, ensuring the latest balloon and aircraft tracking data.

  • Real-Time Data Processing: Retrieves the past 24-hour balloon flight history, filters corrupted data, and interpolates missing timestamps.
  • Weather Integration: Incorporates OpenWeather API data to enhance trajectory predictions based on wind speed and direction.
  • Machine Learning: Uses a Random Forest Regressor to predict future balloon speed & direction based on historical flight data.
  • Airspace Collision Detection: Implements a KD-tree nearest-neighbor search to identify potential conflicts between balloons and aircraft.
  • Interactive Visualization: Displays a real-time map with live balloon trajectories, predicted future paths, and identified risk zones.
  • Automated Updates: Uses a Flask cron job to refresh live data every hour, ensuring continuous tracking.
  • Deployment: Hosted online using FlaskDeploy, making the system publicly accessible with real-time updates.
  • Applications: Enhances aviation safety, aids meteorological research, and improves unmanned balloon navigation.
